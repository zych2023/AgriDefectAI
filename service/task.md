核心目标：面向农作物病虫害识别、农技知识问答、种植建议和田间管理场景，构建智慧农业 AI 系统。系统需包含图像识别模型、农业知识库 RAG、种植决策 Agent、农户端、技术员端和后台管理端。
任务 1：实现病虫害图像识别模块，支持图片上传、预处理、分类/检测、置信度展示和相似样例查看。
任务 2：建设农业知识库，支持作物品种、病虫害图鉴、农事操作、施肥用药说明和常见问答检索。
任务 3：实现种植决策 Agent，调用病害识别、农业数据 SQL Agent、知识检索、农事日历、天气信息、流式语音农技问答和建议生成工具。
任务 4：实现农户端页面，支持拍照识别、病害说明、处置建议、问答咨询、历史记录和反馈。
任务 5：实现后台管理端，支持农户管理、农技专家管理、农田管理、作物管理、病虫害管理和识别记录管理。
1. 需求分析：梳理农户识别、技术员复核、农技资料维护和后台管理流程。
2. 架构设计：后端提供图片上传、模型推理、知识库检索、Agent 编排和业务管理接口。
3. 图像模型实现：完成数据预处理、模型训练或调用、推理接口、置信度输出和结果解释。
4. RAG 实现：完成农技资料解析、检索、引用展示和问答生成。
5. Agent 实现：封装识别结果读取、农业数据 SQL Agent、知识检索、农事建议、天气查询、流式语音农技问答和复核提交工具。
6. 前端实现：完成图片识别、智能问答、农事建议、历史记录和结果详情页面。
7. 后台实现：完成农户、农技专家、农田、作物、病虫害和识别记录等业务对象管理。
8. 系统联调：完成从图片上传到识别结果、建议生成和后台复核的闭环。
